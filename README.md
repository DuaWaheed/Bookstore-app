Book Store App – eProject
1. Introduction

The Book Store App is a comprehensive mobile application designed to simulate a real-world book retail environment. With the rapid evolution of the book industry and the rise of e-commerce, this app bridges the gap between traditional bookstores and modern digital platforms. It allows users to explore, purchase, and manage books seamlessly while providing administrators with full control over the catalog, users, and orders.


2. Functional Requirements
User Features

Registration & Authentication: Sign up using email or social media; secure login/logout; password reset.

Book Catalog: Browse books with titles, authors, genres, cover images, descriptions, and prices; access bestsellers and new arrivals.

Search & Filters: Search by title, author, or genre; filter by price, popularity, and release date.

Profiles: Update profile info, shipping addresses, and payment methods.

Shopping Cart: Add/remove books, update quantities, and view total price.

Ratings & Reviews: Rate and review books; like other users’ reviews.

Order Management: View order history, track delivery, and check order status.

Wishlist: Save books for future purchase.

Admin Features

Manage book listings, user accounts, and orders.

Add, update, or remove books from the catalog.

3. Non-Functional Requirements

Responsiveness: Interactions respond within 1-2 seconds.

Loading Time: Minimized for quick access.

User Interface: Intuitive, mobile-friendly design.

Accessibility: Clear fonts, legible UI elements, easy navigation.

Error Handling: Robust error messages and graceful handling of unexpected events.

Scalability: Designed to handle increased user traffic and future expansions.

Security: Only registered users can access certain features; secure authentication.


Database: Firebase

5. Setup & Run Instructions

To run the Book Store App on your system, follow these steps:

1. Clone the Project

2. Set Up Firebase

Go to Firebase Console
 and create a new project.

Add an Android/iOS app in the Firebase project.

connect firebase with your project 

*delete my firebase options.dart * and create yours by adding command on terminal

firebase configure

6. Install Flutter Packages

Run the following command in the project root to install all required packages listed in pubspec.yaml:

flutter pub get

7. Run the Application

by writing command on terminal

flutter run

The app should now build and run successfully with Firebase integration enabled.

8. Usage

Users can register, browse books, add to cart, place orders, and rate/review books.

Admins can manage books, users, and orders from the admin panel.

The app is designed for scalability to add new features easily.

9. Conclusion

This Book Store App is a fully functional, real-life project demonstrating mobile app development, database management, and user experience design. It provides a complete learning experience, bridging the gap between classroom concepts and real-world implementation.
